    Installing Python

    Mac OS X

    • If you have Mac OS X 10.6 or above, you're allready set. To check your version, go to Apple Symbol -> About this Mac. To confirm that you have python installed, simple start terminal (Applications->Terminal) and type 'python'. That should start the python interpreter. If you recieve command not found, you need to install python.
    • Otherwise, install Homebrew and then run 'brew install python' in your terminal.

    Linux

    • Simply run your package manager (typically either apt-get or yum) install python
