    About the Workshop

    hack@uchicago, in collaboration with the Masters Program in Computer Science, will be offering an introductory workshop on Python on Saturday, November 8th. There will be two sessions, one at 9am-1pm and another at 2pm-6pm. Both sessions are identical.

    There is limited space for this workshop; if you plan to attend, please register here. Please note that this workshop is open only to UChicago students, staff, faculty, and alumni.

    This workshop provides a rapid introduction to Python, and assumes that participants already know at least one programming language, and are comfortable using a UNIX command-line interface. This workshop is not suitable for students with little or no programming experience. If you are unsure of whether you meet the prerequisites for this workshop, please see our Prerequisites page.

    If you are attending the workshop, please make sure you follow the steps listed in our Before the Workshop page. In particular, we will assume that you have remote access to a CS Linux login machine (which already has Python installed on it), or that you have installed Python on your laptop and are comfortable using the command-line interface on your operating system.
