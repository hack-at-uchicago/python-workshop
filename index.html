    About the Workshop

    hack@uchicago will hold an Introduction to Python workshop on October 8th, from 5pm to 8pm, in Stuart 101. There is limited space for this workshop; if you plan to attend, please register here.

    If you are attending the workshop, you will need to bring a laptop. You also need to install Python on your laptop before the workshop.

    Installing Python

    Mac OS X

    • If you have Mac OS X 10.6 or above, you're allready set. To check your version, go to Apple Symbol -> About this Mac
    • Otherwise, install Homebrew and then run 'brew install python' in your terminal.

    Linux

    • Simply run your package manager (typically either apt-get or yum) install python

    Windows

    • Documentation for Windows Installation

    IPython

    • Optionally, you should also install the IPython shell. Their page provides detailed installation instructions. Please note that, if you are unable to install IPython, you will still be able to follow the examples and exercises in the workshop. IPython provides some tools that you may find useful, but which are not strictly required for this workshop.

    Trouble?

    • If you encounter any trouble installing Python on your laptop, please plan to arrive early. TA's will be available in Stuart 101 starting at 4:30 to help with installation issues.
